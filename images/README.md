# Изображения героев

Поместите сюда изображения героев Dota 2.

Список необходимых файлов:
- axe_full.png
- sven_full.png
- drow_ranger_full.png
- pudge_full.png
- riki_full.png
- crystal_maiden_full.png
- juggernaut_full.png
- lina_full.png

Вы можете скачать их с:
- https://www.dota2.com/heroes/ (официальный сайт)
- https://dota2.fandom.com/wiki/Heroes (wiki)
- Или использовать любой другой источник изображений героев

Альтернативно, можно скачать используя curl или wget:
```bash
cd images/heroes
curl -O https://cdn.dota2.com/apps/dota2/images/heroes/axe_full.png
curl -O https://cdn.dota2.com/apps/dota2/images/heroes/sven_full.png
curl -O https://cdn.dota2.com/apps/dota2/images/heroes/drow_ranger_full.png
curl -O https://cdn.dota2.com/apps/dota2/images/heroes/pudge_full.png
curl -O https://cdn.dota2.com/apps/dota2/images/heroes/riki_full.png
curl -O https://cdn.dota2.com/apps/dota2/images/heroes/crystal_maiden_full.png
curl -O https://cdn.dota2.com/apps/dota2/images/heroes/juggernaut_full.png
curl -O https://cdn.dota2.com/apps/dota2/images/heroes/lina_full.png
```
